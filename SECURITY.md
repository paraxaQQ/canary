# Security policy

## Supported versions

Security fixes are applied to the latest `0.2.x` release line.

## Reporting a vulnerability

Use GitHub's private vulnerability reporting for this repository. Include reproduction steps,
affected inputs or URLs, impact, and any proposed mitigation. Do not open a public issue until a
fix is available.

Reports about detector bypasses are also welcome. State whether the bypass causes a false negative,
a crash, credential exposure, unintended network access, or unbounded resource use.
